4. 使用指令打开指定的程序

例如,我喜欢使用code .来用vscode来打开项目文件.
首先,找到vscode的执行程序.我的是在’/Applications/code.app/Contents/Resources/app/bin/code’.
编辑系统根目录的’.zshrc’,在最后一行的地方,添加alias code='/Applications/code.app/Contents/Resources/app/bin/code'
保存,重新打开终端即可使用指令打开vscode了

7. 挂载nfts格式的U盘

mac 其实可以识别ntfs的U盘,但是不能显示中系统中.
需要先卸载,然后再手动挂载U盘.

  1. 插入ntfs的U盘
  2. 在终端中执行mount,打印已挂载的设备,找到U盘路径. 我目前的路径是’/dev/disk2s1’
  3. 卸载U盘: sudo umount /dev/disk2s1.
  4. 创建一个目录作为U盘的挂载点,用于打开U盘: mkdir ntfs-disk
  5. 重新挂载: sudo mount -t ntfs -o rw,nobrowse /dev/disk2s1 /Users/username/ntfs-disk
    其中-t为’type’,ntfs是格式;-o为’options’,rw表示可读可写;nobrowse表示不需要预览;最后两项是’U盘路径’和’挂载的文件路径’.
    /Users/username/ntfs-disk这个路径时当前系统用户下的文件,需要知道完整的路径,通过pwd获取.
