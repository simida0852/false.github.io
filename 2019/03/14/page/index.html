macbook日常使用相关

macbook日常使用相关

1.安装 ‘brew’

参考网站
打开终端 (command + space, 搜索 ‘terminal’)
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

2. mac个性终端 iterm2

参考网站
直接下载安装就好

3. 给终端加个主题

参考网站
打开终端
sh -c "$(wget https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"
安装完之后,在系统根目录会多一个’.zshrc’配置文件.如果觉得默认的主题不好看,可以换你喜欢的.主题预览
选择主题后,把主题名字复制,修改’.zshrc’中的 ZSH_THEME="robbyrussell"即可.

4. 使用指令打开指定的程序

例如,我喜欢使用code .来用vscode来打开项目文件.
首先,找到vscode的执行程序.我的是在’/Applications/code.app/Contents/Resources/app/bin/code’.
编辑系统根目录的’.zshrc’,在最后一行的地方,添加alias code='/Applications/code.app/Contents/Resources/app/bin/code'
保存,重新打开终端即可使用指令打开vscode了

5. 删除文件夹(非空)

之前试过rmdir命令,但是无法删除非空的文件夹,
后来查了一下,使用rm -rf 文件夹,就可以了.

6. 窗口分布插件

设置每个程序窗口的大小(按照宫格尺寸).
不知道怎么描述,反正看官方文档就OK了.

7. 挂载nfts格式的U盘

mac 其实可以识别ntfs的U盘,但是不能显示中系统中.
需要先卸载,然后再手动挂载U盘.

  1. 插入ntfs的U盘
  2. 在终端中执行mount,打印已挂载的设备,找到U盘路径. 我目前的路径是’/dev/disk2s1’
  3. 卸载U盘: sudo umount /dev/disk2s1.
  4. 创建一个目录作为U盘的挂载点,用于打开U盘: mkdir ntfs-disk
  5. 重新挂载: sudo mount -t ntfs -o rw,nobrowse /dev/disk2s1 /Users/username/ntfs-disk
    其中-t为’type’,ntfs是格式;-o为’options’,rw表示可读可写;nobrowse表示不需要预览;最后两项是’U盘路径’和’挂载的文件路径’.
    /Users/username/ntfs-disk这个路径时当前系统用户下的文件,需要知道完整的路径,通过pwd获取.

    指令集(持续更新)

  6. 查看当前目录完整路径 : pwd
  7. 复制文件(文件夹) : cp -r ./a/ ./b

    让终端命令走代理

    export ALL_PROXY=socks5://127.0.0.1:1080
